== Description ==

This widget lets you put messages in colorful notes to capture visitor's attention. 

Some of the main functionality are:

* You can add many messages as you want
* All the messages will appear one after another
* Interval of time set by you
* You can use expires date function to disable messages automatically
* Choose from 6 colornotes options

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the 'wp_colornotes' folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After you activate the plug, you will see a new option called "WP-ColorNotes" in the "Settings" menu.

= Configuring the Widget =
   
1. Once you have added messages, go to the "Widgets" section in the "Appearance" menu. 
2. Look for the "WP-ColorNote" widget an click the "add" link.
3. When the widget was added to the click on the "edit" link.
4. There, you can configure the time thats one message will stay in the screen before another message appears, and the style of a color note.
5. Press "Save Changes". Done !

= Adding Messages =
1. Go to the "ColorNotes" option in the Settings menu.
2. Click in the "Add New Note"
